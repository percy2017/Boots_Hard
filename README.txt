== Description ==

Boots Hard es un tema de WordPress personalizado construido con Bootstrap, diseñado para ser flexible y fácil de personalizar a través del Personalizador de WordPress.

Características Principales:

*   **Basado en Bootstrap:** Utiliza el framework Bootstrap para un diseño responsive y componentes modernos.
*   **Amplias Opciones de Personalización:** Configura fácilmente la apariencia de tu sitio a través del Personalizador de WordPress (`Apariencia` > `Personalizar`):
    *   **Colores Personalizados:** Define colores primarios, secundarios, de éxito, peligro, advertencia, información, claro y oscuro. Ajusta el color del texto y el fondo del cuerpo. El contraste del texto se ajusta automáticamente.
    *   **Diseño y Ancho:** Controla el ancho del contenido principal (`container` vs. `container-fluid`), el ancho interno de las secciones y la opción de menú de navegación fijo (sticky).
    *   **Tipografía:** Selecciona familias de fuentes para el cuerpo y los encabezados.
    *   **Tamaños de Fuente:** Ajusta el tamaño de la fuente para el cuerpo y encabezados principales.
    *   **Espaciado entre Secciones:** Controla el espaciado vertical entre secciones (`py-0` a `py-5`).
    *   **Resetear Estilos:** Restaura todas las opciones de estilo del tema a sus valores por defecto.
*   **Estructura de Secciones Modulares:** Utiliza partes de plantilla (`template-parts/sections/`) para construir la página de inicio.
*   **Navegación Personalizada:** Incluye un Navwalker personalizado de Bootstrap.

== Installation ==

1.  En tu panel de administración de WordPress, ve a `Apariencia` > `Temas`.
2.  Haz clic en `Añadir nuevo`.
3.  Busca "Boots Hard" si está en el directorio de WordPress.org, o haz clic en `Subir tema` si tienes el archivo `.zip`.
4.  Si subes el tema, selecciona el archivo `.zip` descargado y haz clic en `Instalar ahora`.
5.  Una vez instalado, haz clic en `Activar`.

== Frequently Asked Questions ==

= ¿Cómo accedo a las opciones de personalización del tema? =

Ve a `Apariencia` > `Personalizar` en tu panel de administración de WordPress. Busca el panel "Estilos del Tema (nuestro)".

= ¿El tema es responsive? =

Sí, Boots Hard está construido sobre Bootstrap, que es un framework responsive por defecto.
